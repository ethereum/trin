# Introduction to Trin

> trin: a fast Rust client for Ethereum access via the Portal Network.

Trin acts as a json-rpc server, like an Ethereum client.

Unlike an Ethereum client, trin has:
- Nearly instant sync
- Low CPU & storage usage
